文字排版 Typography

很多设计师喜欢用英文，因为中文排版真的不容易搞。中文字符多、不同字符的笔画差异也很大，使用英文时高大上的设计稿，替换成中文以后， 可能会大打折扣。网页设计中，针对中文排版的研究很少，没有太多现成的杰伦供参考。Xiaoi UI 只能根据一些经验，在字体设置、字号上做一些 更适合中文的设置。Xiaoi UI 主要使用非衬线字体中文默认微软雅黑，英文默认Arial.

实例 :

  • 标准字

  • 标准字

    大小: 18px 行间距: 24px 用于内容一级标题，例如左边栏主标题，搜索文章的主标题

模态窗口

有2情形，一种显示界面已有的dom元素，另一种是提示的形式

实例 :

                            
            $('#modelDomBtn').click(function (){
                    ui.model('.model-dom');  // .model-dom 是页面已存在的dom元素
            });

实例 :

                            
            // 传入数组，分别为提示的信息，和确认之后执行的函数
            $('#modelArrayBtn').click( function (){
                    ui.model( [ '弹出提示信息', function  (){
                            alert( '点击确认执行的函数' );
                    } ] );
            });
